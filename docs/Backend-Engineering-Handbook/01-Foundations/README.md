# 01 - Foundations

This section introduces the core concepts every backend developer needs before building applications.

The focus is on understanding *why* each concept exists rather than memorizing syntax.

Topics include object-oriented programming, dependency injection, REST APIs, LINQ, asynchronous programming, and the fundamental ideas that appear throughout modern C# and ASP.NET Core development.

These concepts are referenced throughout the rest of the handbook and provide the foundation for everything that follows.
