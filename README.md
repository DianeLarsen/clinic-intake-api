# Clinic Intake API

## Purpose

A learning project to build a backend API using ASP.NET Core, C#, dependency injection, repository patterns, and Azure cloud services.

## Current Architecture

ASP.NET API
→ Service Layer
→ Repository Layer
→ In-Memory Storage

## Technologies

- C#
- ASP.NET Core
- Swagger
- Git
- GitHub

## Planned Features

- CRUD API endpoints
- Database integration
- Azure deployment
- Authentication
